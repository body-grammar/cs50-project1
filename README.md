# Wiki Encyclopedia

A Wikipedia-style encyclopedia web application built as part of Harvard's CS50W: Web Programming with Python and JavaScript.

## What it does

Allows users to create, edit, search, and delete wiki entries. Content is rendered from Markdown into formatted HTML pages, similar to how Wikipedia displays articles.

## Features

- Create new wiki entries
- Edit existing entries
- Search for entries by title or content
- View a random entry
- Markdown-to-HTML rendering for clean formatting

## Built with

- Python (Django)
- HTML/CSS
- Markdown2 (for converting Markdown to HTML)

## What I learned

This project helped me understand server-side routing in Django, handling forms, and working with file-based content storage instead of a traditional database.

## How to run it

\`\`\`
python manage.py runserver
\`\`\`
Then visit `http://127.0.0.1:8000` in your browser.
